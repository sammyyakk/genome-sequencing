# 📄 Paper 8: Deep Learning: New Computational Modelling Techniques for Genomics
*   **Journal / Conference**: Nature Reviews Genetics (2019)
*   **DOI**: `10.1038/s41576-018-0026-1`

---

## 🔬 Literature Review Table

| Evaluation Field | Researcher Analysis |
| :--- | :--- |
| **Problem Addressed** <br>*(1-2 lines)* | _[With traditional methods like Sanger sequencing and karyotyping , it is very difficult to treat the rare genetic disorders , which leaves around 50% of patients undiagnosed from these genetic disorders causing significant diagnostic delays.]_ |


| **Methodology & Approach** <br>*(2-3 lines)* | _[The researchers proposed the framework on Whole Genome Sequencing. They used the Swegen dataset , which contains information about gender, age, ethnicity, etc. and then cleaned and processed the dataset by applying some normalizing filters. Then extracted key genetic markers and used a Random Forest classifier trained on these features to detect the genetic diseases.]_ |


| **Key Results & Findings** <br>*(2-3 lines)* | _[This framework achieved about accuracy of 97% and about 95% precision. It outperformed other models like logistic regression(89%), SVM(90%) and KNN(94%). Completed around 9.85 seconds quicker than the other methods , and thus successfully identified the 10 unique disease markers accross various categories like neurological , cardiovascular , etc.]_ |

| **Limitations Identified** <br>*(1-2 lines)* | _[This framework was validated only on Swedish population, with a limited feature set , so cross population generalizability remains unexplored. ]_ |


| **Future Scope** <br>*(2-3 lines)* | _[They recommended to expand the dataset to include larger , more diverse population to improve generalizability , along with the machine learning models and enhancing feature selection. Also emphasized on the need for validation studies with real world clinical data. ]_ |


| **Project Relevance** <br>*(1-2 lines)* | _[The paper provides a clear method for combining whole genome sequencing with machine learning to significantly improve rare disease detection. ]_ |
