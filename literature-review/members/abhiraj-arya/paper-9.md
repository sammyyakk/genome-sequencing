# 📄 Paper 9: Highly Accurate Protein Structure Prediction with AlphaFold
*   **Journal / Conference**: Nature (2021)
*   **DOI**: `10.1038/s41586-021-03819-2`

---

## 🔬 Literature Review Table

| Evaluation Field | Researcher Analysis |
| :--- | :--- |
| **Problem Addressed** <br>*(1-2 lines)* | _[The paper addresses long-standing problem of predicting a protein's 3D structure from its amino acid sequence. Methods like X-Ray crystallography , NMR spectroscopy and cryo-EM are still slow and expensive for this. ]_ |


| **Methodology & Approach** <br>*(2-3 lines)* | _[The researchers presented AlphaFold, a deep learning model that uses protein's amino acid sequence to predict the 3D positions of all heavy atoms.This model has two main parts; a trunk that processes sequence data using attention and non attention method and a structure module that builds the 3D structure using rotation and translation for each part of protein.]_ |


| **Key Results & Findings** <br>*(2-3 lines)* | _[Alphafold performed very well in the CASP14 challenge with the highest level of accuracy. It generated about 192 structures during its processing , with the continuous improvement in every predicted shape.]_ |


| **Limitations Identified** <br>*(1-2 lines)* | _[The accuracy dropped when the number of similar sequences used is below is 30. AlphaFold also struggles with proteins where parts of the same chain dont interact much.]_ |


| **Future Scope** <br>*(2-3 lines)* | _[AlphaFold's approach can be used to predict structures of complexes made of different proteins.As the sequence data increases beyond 100 sequences, the model continues to improve slightly .]_ |


| **Project Relevance** <br>*(1-2 lines)* | _[As AlphaFold offers a fast and accurate way to predict the structure of protein, it can be used in drug development, enzyme design and understanding the protein function.]_ |
