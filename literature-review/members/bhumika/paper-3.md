# 📄 Paper 3: An integrated encyclopedia of DNA elements in the human genome
*   **Journal / Conference**: Nature (2012)
*   **DOI**: `10.1038/nature11247`
*   **Link**: [Nature](https://www.nature.com/articles/nature11247)

---

## 🔬 Literature Review Table

| Evaluation Field | Researcher Analysis |
| :--- | :--- |
| **Problem Addressed** <br>*(1-2 lines)* | _[Most of the human genome doesn't code for proteins, and its function was largely unknown scientists needed a systematic map of what the rest of the DNA actually does.]_ |
| **Methodology & Approach** <br>*(2-3 lines)* | _[A massive consortium ran 1,640 experiments across 147 cell types, using techniques like ChIP-seq (finds where proteins bind DNA), DNase-seq (finds "open" accessible DNA), and RNA sequencing, then integrated all this data computationally.]_ |
| **Key Results & Findings** <br>*(2-3 lines)* | _[80.4% of the genome shows some biochemical activity in at least one cell type — far more than the ~1.2% that codes for proteins. They mapped ~400,000 enhancer-like regions and ~70,000 promoter-like regions (DNA "switches"), and found that disease-linked DNA variants (from GWAS) cluster heavily in these newly identified functional, non-coding regions.]_ |
| **Limitations Identified** <br>*(1-2 lines)* | _[The project didn't test all cell types or all proteins, so the true amount of functional DNA is likely underestimated; "biochemical activity" doesn't always mean a real, organism-level function means some signals might be biologically irrelevant noise.]_ |
| **Future Scope** <br>*(2-3 lines)* | _[Plans to expand to more cell types and factors, and combine with related projects (like the Roadmap Epigenomics Project) to build a more complete map of how genes are regulated, especially relevant for understanding disease-causing variants outside genes.]_ |
| **Project Relevance** <br>*(1-2 lines)* | _[Provides essential background for genome sequencing and shows that interpreting DNA isn't just about finding mutations in genes; non-coding regulatory regions (where most disease variants actually sit) matter just as much, which should inform how our project interprets sequencing results.]_ |
