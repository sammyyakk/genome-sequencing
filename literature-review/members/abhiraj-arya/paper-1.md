*   **Journal / Conference**: Nature Reviews Genetics (2019)
*   **DOI**: `10.1038/s41576-019-0122-6`
*   **Link**: [Nature Reviews Genetics](https://www.nature.com/articles/s41576-019-0122-6)

## 🔬 Literature Review Table

| Evaluation Field | Researcher Analysis |
| :--- | :--- |
| **Problem Addressed** <br>*(1-2 lines)* | _[As the Genomics data is flooded with lots of data but it lacks the technical skills required to analyze it effectively using neural networks. So the paper looks at how deep learning can handle better than the older , hand-tuned machine learning methods.]_  |

| **Methodology & Approach** <br>*(2-3 lines)* | _[The authors went through existing deep learning work in genomics and grouped it by architecture type as fully connected, CNN(convolutional neural network), RNN(Recurrent neural network), graph convolutional and autoencoder and then compared how each performs against machine learning approaches on real problems of genomics.]_ |

| **Key Results & Findings** <br>*(2-3 lines)* | _[One clear takeway is that CNN have become the standard for pattern discovery and variant effect scoring due to their ability to detect positional patterns. On the unsupervised side VAE and adversial networks are particularly useful when helping with dimensionality reduction, synthetic data correction and phenotype prediction.]_ |

| **Limitations Identified** <br>*(1-2 lines)* | _[Deep Learning models are hard to explain biologically since they learn complex, layered features on their own and also expensive to train and can skip over biologically important features if the data is not set up right, since many datasets are too small for these models to perform really well.]_ |

| **Future Scope** <br>*(2-3 lines)* | _[Generative models may soon be used to create synthetic genomes for benchmarking without exposing patient data. Federated learning and modelling is seen as a practical solution for improving the data piracy , synthetic data generation. Deep learning is soon going to be crucial tool in genomics.]_ |

| **Project Relevance** <br>*(1-2 lines)* | _[Deep learning reduces the manual work of spotting patterns and generally outperforms older Machine Learning approaches and helps to determine which architecture is best for different types of data types, where interpretation might be misleading and which validation methods to use.]_ |
